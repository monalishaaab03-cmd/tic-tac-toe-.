 Tic Tac Toe in C

 Description :
A simple Tic Tac Toe game in C for two players.
This project demonstrates basic programming concepts like arrays, loops, and functions in C.

 Features :
- 2 Player Game
- Win Detection
- Draw Detection
- Input Validation

 Technologies Used :
- C Programming Language

 How to Run :
1. Compile the code:
   gcc tic_tac_toe.c -o game
2. Run the program:
   ./game
